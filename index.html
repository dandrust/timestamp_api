Timestamp API

Pass a natrual language date or unix timestamp as a parameter. Returns JSON file with natural langauge date and unix timestamp.


Example input:

https://dandrust-timestamp-api.herokuapp.com/January%2017,%201988

https://dandrust-timestamp-api.herokuapp.com/569376000000


Example output:

{"natural":"January 17, 1988", "unix":"569376000000"}



Coded by Dan Drust for completion of Free Code Camp's API timestamp microservice code challenge, August 2016
